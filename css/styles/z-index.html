z-index.sass

The stacking order of HTML elements.

By default:

  • child elements are on top of their parents
  • for siblings, if it's later in the HTML code, it's on top of its siblings

z-index only works if you have position set to either relative, absolute, or fixed..

All values are relative, meaning that an element
