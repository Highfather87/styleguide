lists.sass

For lists of things, when you use #each model as |thing| basically.

There are 9 lists right now:

  • articles: the /basket articles
  • boutiques: the /boutiques grid
  • categories: the /snap categories
  • members: the /our-team members
  • photos: the /products/:id image gallery
  • products: the /shop and /me/wishlist list of products
  • recents: the recently viewed products
  • similars: the similar products
  • rows: the /landing and /discover entries

The naming convention is:

  • list-things in plural for the container
  • list-thing in singular for the individual items

So for example, the list of products looks like this:

{% highlight html %} {% assign opening = '{{#each products as |product|}}' %} {% assign closing = '{{/each}}' %}
{{ opening }}
{{ closing }}
{% endhighlight %}
