buttons.sass

All the buttons of the website.

You don't write button button-green button-large in your HTML, but you just write button-get-shopping and then you compose your button in Sass.

{% highlight sass %} %button %button-default %button-primary %button-secondary %button-green %button-blue =button-tall =button-grande =button-venti {% endhighlight %} {% highlight html %} the main button styles, for every button you call these styles with @extend %button the different colours called with @extend %button-default the different sizes called with +button-tall rest of the css is composing each individual button {% endhighlight %}
