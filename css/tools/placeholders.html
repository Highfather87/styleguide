placeholders.sass

Sass placeholders are a way to regroup common styles used by multiple selectors. They can be unpredictable.

You define a placeholder with the percentage sign %placeholder.

You call a placeholder with the @extend rule @extend %placeholder.

They are more complicated than mixins:

  • they don't have parameters
  • they do prevent the same code from being repeated

The only real benefit is that it's really DRY.

Example:

{% highlight sass %} %control cursor: pointer width: 100% white-space: nowrap .checkbox @extend %control .input @extend %control .radio @extend %control .select @extend %control {% endhighlight %}

will result in:

{% highlight css %} .checkbox, .input, .radio, .select { cursor: pointer width: 100% white-space: nowrap } {% endhighlight %}

You'll notice the original %control is not present in the final result. It's only used in the original Sass to share common code between the 4 selectors.
